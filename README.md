# pyweek31
Cyber Cop Stealing Powers And Stuff! Wow!

## Setup

```
python -m venv venv
```

Activate the virtual env.

```
pip install -r requirements.txt
```

Start the game
```
python run_game.py
```

## Style

For style we use `black` a tool to auto format the code, use it with:
```
black .
```
